  <p align="center">
    Weather affects every single person on this planet every day. It can affect your whole day as some activities may not be suitable to do in certain weather. That's why it is important to have the ability to check the current and upcoming weather.<br><br>
  At the moment this web app can only display current weather. The OpenWeatherMap free API only supports 3-hour forecast and current weather, so I'm only able to add a short forecast display. This will come at a later date.

## About The Project

[![Product Name Screen Shot][product-screenshot]](https://example.com)

WeatherApp is the second iteration of my portfolio. I wanted to make an app that utilizes an API, because knowing how to use APIs and how to fetch and handle data is something you will certainly come across when working as a software developer. Also, the recent searches feature made me learn how localstorage works on browsers, which is a very handy tool to store small bits of data when the browser is closed.

## Test via local repo

To get a local copy up and running follow these simple example steps.

### Prerequisites

Install latest version of npm
* npm
  ```sh
  npm install npm@latest -g
  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/Larppa1/WeatherApp
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
